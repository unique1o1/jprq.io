# jprq.io

## Get Your Localhost Online and HTTPS


#### **Note** this server doesn't work with the original [jprq](https://github.com/azimjohn/jprq) client. Please use the client [jprq-go-client](https://github.com/unique1o1/jprq-go-client) instead.


`github.com/unique1o1/jprq.io/`
- [Build](#pkg-build)
- [Overview](#pkg-overview)



## <a name="pkg-build">Build</a>
```bash
go build .
./jprq.io
```
### Note
- To get the server working, add an `A Record` in you DNS pointing your wildcard subdomain to your server's IP address.
